Google Analytics Test Page

This page is used to test Google Analytics tracking.

If you can see this page, the tracking code is loaded.

Test Instructions:

  1. Open this page in your browser
  2. Open Developer Tools (F12)
  3. Go to Console tab
  4. Look for gtag calls
  5. Check Network tab for requests to googletagmanager.com

Expected Results:

  • Console should show gtag function calls
  • Network tab should show requests to Google Tag Manager
  • Google Analytics should show real-time data
